### Vue多页签组件

| 组件属性                           | 说明                                            | 默认值 |
| ---------------------------------- | ----------------------------------------------- | ------ |
| keepAliveComponentInstance（必需） | keep-alive的控件实例对象，获取方式可查看App.vue | null   |
| blankRouteName                     | 空白页面的路由名称                              | blank  |
